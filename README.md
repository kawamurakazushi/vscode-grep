# vscode-grep README

## Features

Incremental grep extension for Visual Studio Code.

## Git Grep (grep.git)

![screenshot](images/screenshot.gif)

## Extension Settings

| Setting            | Description                                         | Type    | Default Value |
| ------------------ | --------------------------------------------------- | ------- | ------------- |
| `grep.git.preview` | Enable / Disable preview when the item is selected. | boolean | false         |

## Commands

```json
"commands": [
  {
    "command": "grep.git",
    "title": "Grep: Git"
  }
]
```
